# Creating an Annotated Bibliography — Interactive Presentation

**Tulane University Libraries — Howard-Tilton Memorial Library**
Presented by Kay P. Maye

---

## Overview

This interactive HTML presentation teaches undergraduate and graduate students how to create an annotated bibliography. It covers the complete process from understanding what an annotated bibliography is, through finding and evaluating sources, to writing and self-assessing annotations. The presentation includes three hands-on activities with built-in feedback, a Boolean search string generator, and tools for students to email or download their work.

## Presentation Structure

The presentation contains **12 slides** progressing from concept introduction through guided practice:

| Slide | Title | Content |
|-------|-------|---------|
| 0 | **Title Slide** | Roadmap overview (Learn → Anatomy → Practice) |
| 1 | **What Is an Annotated Bibliography?** | Side-by-side comparison of a regular bibliography vs. annotated entry |

| 3 | **Types of Annotations** | Comparison table: Summary, Evaluative, and Reflective annotations with example phrases |
| 4 | **Anatomy of an Entry** | Full worked example using Twenge & Campbell (2018) with color-coded Summary → Evaluation → Reflection sections |
| 5 | **Citation Styles: APA, MLA & Chicago** | Style comparison table, Zotero and Purdue OWL resources |
| 6 | **Writing Strong Annotations** | 3-part formula (Summarize → Evaluate → Reflect) + common mistakes to avoid in two-column layout |
| 7 | **Finding Sources** | Library Search, subject databases, Google Scholar setup, Boolean search generator, search tips |
| 8 | **Activity 1: Evaluate an Annotation** | Compare weak vs. strong annotations for Orben & Przybylski (2019); checklist with automated feedback |
| 9 | **Activity 2: Write Your Own** | Scaffolded writing workspace (citation → summary → evaluation → reflection → combined paragraph) |
| 10 | **Activity 3: Quality Checklist** | 15-item self-assessment checklist covering citation, summary, evaluation, reflection, and overall quality |
| 11 | **Summary & Next Steps** | Key takeaways, action items, export tools, contact information |

## Interactive Features

### Navigation
- **Top navigation bar** with Previous/Next buttons, dropdown slide selector, and page indicator
- **Keyboard shortcuts:** Arrow keys and Space to advance, PageUp/PageDown, Home/End
- **Progress bar** at the bottom of the screen
- **Activity tracker** (top-right) showing completion status for all three activities

### Boolean Search Generator (Slide 7)
- Add multiple concept rows connected with **AND**
- Add synonym terms within concepts connected with **OR**
- Add exclusion concepts with **NOT**
- Multi-word terms automatically wrapped in quotes for exact-phrase searching
- Copy generated search string to clipboard
- Concepts and synonyms update the preview in real-time

### Activity 1: Evaluate an Annotation (Slide 8)
- Side-by-side comparison of a weak and strong annotation for the same source
- Four-item checklist identifying what the weak annotation is missing
- Automated feedback (correct, partial, or incorrect) based on selections
- Open-ended reflection textarea

### Activity 2: Write Your Own (Slide 9)
- Scaffolded writing with separate fields for citation, summary, evaluation, and reflection
- Color-coded labels matching the 3-part formula taught in Slide 6
- Placeholder prompts with sentence starters
- Final combined paragraph field for the complete annotation

### Activity 3: Quality Checklist (Slide 10)
- 15-item self-assessment organized into four categories: Citation (3 items), Summary (4 items), Evaluation (3 items), Reflection (2 items), Overall Quality (3 items)
- Checkboxes with strikethrough styling on completion

### Response Collection & Export (Slide 11 + floating panel)
- **Copy All Responses** — copies all activity work to clipboard as formatted text
- **Download as Text** — saves responses as a `.txt` file
- **Print / Save as PDF** — browser print dialog with print-optimized CSS
- **Email My Responses** — floating panel (bottom-right button) that opens the user's email client with all responses pre-formatted in the body, addressed to the student's own email
- Response count badge updates in real-time as students complete activities

## How to Use

### For Instructors / Presenters
1. **Open `annotated_bibliography_presentation.html`** in any browser. No internet required after initial load.
2. **Navigate** using arrow keys, the top nav bar, or the dropdown selector.
3. **Present** in full-screen mode (F11 in most browsers) for the cleanest display.
4. **Pause on activities** (Slides 8–10) to give students time to work through each exercise.
5. **Direct students to the email panel** (bottom-right button) at the end so they can save their work.

### For Students (Self-Paced)
1. Open the file in any browser and work through slides in order.
2. Complete all three activities — the activity tracker in the top-right shows your progress.
3. On the final slide, use "Copy All Responses" or "Email My Responses" to save your work.
